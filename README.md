HonKit plugin for customize script
================

## Usage

Add the plugin to your `book.js`:

```
module.exports = {
    "plugins": [ "@ehimemikan/add-js" ]
}
```

## Options
```
module.exports = {
    "pluginsCofig": {
        "add-js": {
            "js": [
                "./your/custom.js"
            ]
        }
    }
}
```
